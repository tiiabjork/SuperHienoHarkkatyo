# SuperHienoHarkkatyo

**Super hienossa harkkatyössä** luodaan To Do -lista, jonka avulla jokainen kiireinen ja vähemmän kiireinen ihminen voi pitää listaa tehtävistä asioista. Erittäin hyvä apuväline, jos tekemistä on paljon.

Ohjelmassa käyttäjä voi lisätä uusia To Do -listoja, jotka näkyvät kaikki listattuna omassa valikossaan. Täältä yhden listan valitessaan ilmestyvät sen listan tekemättömät ja tehdyt tehtävät omiin ruutuihinsa, joista niitä voi valita, ja niitä voi merkitä tehdyiksi ja tekemättömiksi.

Käyttäjä voi lisätä uusia tehtäviä sillä hetkellä valittuna olevaan listaan, jolloin lisäämisen jälkeen ne näkyvät heti tekemättömien tehtävien ruudussa.

Käyttäjällä on mahdollisuus poistaa sekä tehtäviä, että listoja.
